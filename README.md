# Timezone Clock Documentation

## Overview
This timezone clock application allows users to view the current time across various time zones. It's designed to be user-friendly and efficient.

## Features
- Display of current time for multiple time zones.
- Calculation of time differences between locations.
- User interface that updates in real time.

## Installation
To install the timezone clock application:
1. Clone the repository:  
   ```bash
   git clone https://github.com/rr4743484/clock-app.git
   ```  
2. Navigate to the project directory:  
   ```bash
   cd clock-app
   ```  
3. Install dependencies:  
   ```bash
   npm install
   ```  

## Usage
After installation, run the application:
```bash
npm start
```

Navigate to `http://localhost:3000` to view the application in your browser.
